# 神经网络深度学习 实践课程

本仓库为BUPT 2023年神经网络与深度学习实践课程的Task代码仓库，共五个子任务，包含了完整的代码和实验报告：
- 深度学习基础(回归问题&分类问题)
- 计算机视觉基础
- 深度自编码器(MINST&All_Dogs)
- 自然语言处理基础(Transformer)
- 生成模型基础(VAE&GAN)
